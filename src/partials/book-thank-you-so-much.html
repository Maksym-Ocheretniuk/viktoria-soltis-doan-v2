Thank You So Much!

This is a wonderful book about the value of freedom and true friendship.

The main character is very lonely, he has no friends at all. One day he finds a cage with an incredibly beautiful yellow bird. The boy decides that it will be his friend, a friend in a cage. But is it possible to be friends if held against one's will?

In communication with the wise bird, the boy learns about true friendship.

But what will happen next? Did the boy understand the lesson? You will learn about this from the book.♥

The board books are told in simple sentences, perfect for reading aloud to babies and toddlers.

A wonderful addition to any home or school library! Please enjoy!♥
